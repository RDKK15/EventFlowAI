# Architectural Decision Records (ADR)

This directory contains the historical record of major architectural decisions made during the development of BizPart AI.

## Purpose

ADRs explain **why** important architectural decisions were made.

Unlike the Constitution, which defines permanent principles, ADRs capture the context, alternatives, trade-offs, and consequences behind individual decisions.

## Lifecycle

```
Proposed
    ↓
Accepted
    ↓
Implemented
    ↓
Superseded (optional)
```

Historical ADRs are never deleted.

## Naming Convention

ADR-001 BusinessUser Membership.md

ADR-002 Progressive AI Onboarding.md

ADR-003 BusinessContext.md

...
